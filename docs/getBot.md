# get-bot Command
* **Description:** Used to provide information on the specified bot fetched from the Paradise Bots API
* **Aliases:** `api.botinfo`
* **Example:** api.get-bot <@Bot>

---

###### Arguements
| Arg           | Description   | Required |
| ------------- |:-------------:| -----:|
| bot. | Mention of the bot to get info for | true  |
