# get-bot Command
* **Description:** `Used to provide information on the specified bot fetched from the Paradise Bots API`

* **Aliases:** `api.botinfo`

* **Example:** `api.get-bot <@Bot>`

---

### Arguements
* **Arg:** bot

* **Description:** A mention of the bot you want to get info for

* **Required:** true
