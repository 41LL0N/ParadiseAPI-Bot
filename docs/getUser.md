# get-user Command
* **Description:** `Used to provide information on the specified user fetched from the Paradise Bots API`

* **Aliases:** `api.userinfo`

* **Example:** `api.get-user <@User>`

---

### Arguements
* **Arg:** user

* **Description:** A mention of the user you want to get info for

* **Required:** true
