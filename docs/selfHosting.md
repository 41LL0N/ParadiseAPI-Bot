* Bot Versions
- [x] v0.0.1

* API Versions
- [x] v1.10.0

## Host Provider(s)
We personally use [Heroku](https://heroku.com/) which is a good (and free with 1000 hours/month) option
But you may also host the bot locally via your terminal/command console.
